🔐 Projeto Login – Formulário de Autenticação

Este repositório apresenta um projeto de formulário de login desenvolvido com HTML5 e CSS3, com foco na criação de uma interface simples, organizada e responsiva.

O projeto inclui campos de entrada para usuário e senha, utilizando recursos nativos do HTML para validação básica, como obrigatoriedade de preenchimento e tipos de input.

Também foram aplicados conceitos de estilização com CSS, buscando um design limpo e uma boa experiência visual para o usuário.

Este projeto representa a prática na construção de formulários e na organização de interfaces, sendo uma base importante para futuras implementações com validação avançada e integração com back-end.

🚀 Tecnologias utilizadas: HTML5 e CSS3
📚 Foco: Formulários, estruturação e estilização
💡 Objetivo: Praticar a criação de interfaces funcionais e bem organizadas
